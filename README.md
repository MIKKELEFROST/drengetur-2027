# Drengetur · Sommer 2027

En lille flersides hjemmeside der pitcher 6 destinationer til en drengetur i sommeren 2027 — for 6-8 mand på 25, der vil have hygge, fest og gode oplevelser.

## Steder
- **Split** (Kroatien) — bådliv og øhop
- **Barcelona** (Spanien) — storby + strand + fest
- **Lissabon** (Portugal) — sol, surf og billig gadefest
- **Budapest** (Ungarn) — ruin bars og budget-fest
- **Mallorca** (Spanien) — charter-nemt med pool-villa
- **Ksamil** (Albanien) — karibisk-blåt vand til value-pris

## Struktur
- `index.html` — forside med kort, filtre og sammenligningstabel
- `split.html`, `barcelona.html`, `lissabon.html`, `budapest.html`, `mallorca.html`, `ksamil.html` — én side pr. destination
- `styles.css` — fælles styling

Ren statisk HTML/CSS — kan hostes direkte via GitHub Pages.

## Noter
Priser er ca. pr. mand for en uge inkl. fly og delt overnatning (8 personer deler) og er kun vejledende estimater for sommer 2027.
